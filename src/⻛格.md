⻛格名称：Modern Soft-Neo UI with Character Illustrations
整体感觉：明亮、柔和、轻量、精致、亲和、带⼀点轻卡通感的专业界⾯。
1. ⾊彩⻛格（Color Language）
主⾊：柔紫⾊、淡粉紫、浅薰⾐草紫（类似 #8C7CF0、#C6B9FF ）
辅⾊：柔绿、柔⻩、淡橙作为强调⾊
背景：⼤⾯积浅灰⽩（近⽩）背景
⾼亮：⽤渐变⾊块（紫→粉）作为模块背景
阴影：柔和的浅阴影，⽆⾼对⽐
整体⾊感：温和、轻盈、有“云朵感”与“清晨光线感”
关键词：Soft pastel、Light gradient、Lavender UI、Warm highlights、No harsh contrast
2. 布局⻛格（Layout Language）
典型为三栏布局：
左侧导航（图标化）
中间主要内容（卡⽚ + 信息区）
右侧附加板块或⽇历
常⽤超⼤留⽩、圆⻆卡⽚、多段信息卡⽚
信息分组⾮常清晰：每组信息被包装成独⽴的“柔软卡⽚”
卡⽚内部使⽤极简 icon + ⼤标题 + ⼩副标题 + 轻量进度条
使⽤柔和边界与悬浮感卡⽚，不⽤边框
关键词：Floating Cards、Soft Containers、Large White Space、Three-Column Layout
3. 插画⻛格（Illustration Style）
这是这个界⾯的最⼤特点之⼀。
插画特征：
现代扁平插画（Modern Flat Illustration）
轻微 3D 感，但仍是平⾯向
⼈物：⼤头身、圆润、柔和线条
颜⾊：柔紫系为主，加⼊暖⾊点缀（橙、⻩）
场景：户外、宇宙⽕箭、远⼭、营地等轻松氛围
氛围：放松、可爱、温暖、专业
关键词：Modern flat illustration、Cute minimal characters、Pastel gradients、Playful but professional
